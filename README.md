# 🤖 DevGPT

An AI-powered conversational web application built with the **MERN Stack** and **Google Gemini API**. DevGPT provides a clean ChatGPT-inspired interface where users can create conversations, continue existing threads, and receive AI-generated responses in real time.

---

## ✨ Features

* 💬 Create new AI conversations
* 📝 Persistent chat history
* 📂 Conversation threads
* ⚡ Real-time AI responses
* 📖 Markdown support for AI replies
* 🎨 Modern responsive UI
* 🌙 Dark theme inspired by ChatGPT

---

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/chandan-dev-456/DevGPT.git
cd DevGPT
```

### 2. Install dependencies

#### Frontend

```bash
cd Frontend
npm install
```

#### Backend

```bash
cd ../Backend
npm install
```

---

## 🔑 Environment Variables

Create a `.env` file inside the **Backend** directory.

```env
MONGO_URI=your_mongodb_connection_string
GEMINI_API_KEY=your_gemini_api_key
```

---

## ▶️ Running the Application

### Start the backend

```bash
cd Backend
npm run dev
```

### Start the frontend

```bash
cd Frontend
npm run dev
```

Open:

```text
Frontend : http://localhost:5173
Backend  : http://localhost:3000
```
